About

I'm a full-stack web developer with experience building applications in Rails, Node & Express. I'm currently teaching the basics of web development in Ruby and JavaScript to novice programmers and designing educational materials that keep folks excited about code.

Ever since I can remember, I've been tinkering with code. Last year I jumped in headfirst and began programming in earnest. I work on Linux & Mac machines, am a super huge nerd for education & games, and have lived in NYC for nearly a decade.

JSON Navigator

As a side project, I created a tool to help my students learn to navigate nested data structures. The application allows the user see a nicely formatted version of their JSON data, to hover over the keys and values, and see how to access that data in either Ruby or JavaScript.

Thanks For Being A Friend

This game was created during Brain Jam 2015, a 48-hour game jam focusing on mental health put on by the Brooklyn Gamery. For this project, I taught myself the basics of Phaser.io, a 2D JavaScript/HTML5 game framework. The game itself focuses on cheering up a futuristic Golden Girl character by "attacking" them with cheerful images.
